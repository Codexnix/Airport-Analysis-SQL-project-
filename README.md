# Airport Operations & Passenger Traffic Analysis using SQL

## 📌 Project Overview

This project analyzes airport operations and passenger traffic data using SQL. The objective is to identify passenger trends, airline performance, airport activity, freight movement, and regional traffic patterns through business-focused SQL analysis.

---

## 🎯 Problem Statement

Airports and airlines generate large volumes of operational data. This project transforms raw airport data into actionable insights by analyzing passenger traffic, airline performance, airport activity, and regional trends.

---

## 📂 Dataset

**Dataset:** Airport Operations Dataset

- PASSENGERS
- FREIGHT
- MAIL

- REGION
- ORIGIN
- ORIGIN_CITY_NAME
- ORIGIN_STATE_ABR

## ❓ Business Questions Solved

### 1. Which airlines handled the highest passenger traffic?

### 2. Which origin airports generated the most passengers?

### 3. Which origin cities generated the most passenger traffic?

### 4. Which states contributed the highest passenger volume?

### 5. Which regions generated the highest air traffic?

### 6. What is the average route distance?

### 7. Which airlines transported the highest freight volume?
